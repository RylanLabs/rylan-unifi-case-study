# Fortress Agent Instructions

## Naming Doctrine

### Python Modules
**REQUIRED:** Use underscores, never hyphens
- ✅ `my_module.py`, `data_processor.py`
- ❌ `my-module.py` (causes `SyntaxError` on import)

**Rationale:**
```python
import my_module  # ✅ Valid identifier
import my-module  # ❌ SyntaxError: invalid syntax
```

### Shell Scripts
**REQUIRED:** Use hyphens, include headers
- ✅ `deploy-app.sh`, `backup-db.sh`
- ✅ Shebang: `#!/usr/bin/env bash`
- ✅ Headers: Script, Purpose, Guardian, Date, Consciousness

### All Files
- ✅ EOF newline (W292)
- ✅ LF line endings (no CRLF)
- ✅ No BOM (Byte Order Mark)
- ✅ No trailing whitespace on blank lines

### Enforcement
- Pre-commit hook blocks violations
- Auto-fix available: `./scripts/auto-fix-naming.sh`
- CI validates as safety net
